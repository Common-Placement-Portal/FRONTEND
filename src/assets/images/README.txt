INSTRUCTIONS TO ADD CAROUSEL IMAGES
====================================

Please add the following 4 images to this folder (src/assets/images/):

1. cdac1.jpg - Students in auditorium/classroom (like image 3 you provided)
2. cdac2.jpg - Library or study area (like image 2 you provided)
3. cdac3.jpg - Group discussion or meeting (like image 4 you provided)
4. cdac4.jpg - Computer lab or students working (like image 5 you provided)

The carousel will automatically cycle through these images every 4 seconds.

Image Requirements:
- Format: JPG or PNG
- Recommended size: 600x400 pixels or similar aspect ratio
- File names must match exactly: cdac1.jpg, cdac2.jpg, cdac3.jpg, cdac4.jpg

After adding the images, the carousel on the home page will display them automatically.
